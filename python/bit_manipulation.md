# Bitwise Operators

- You can use left, right shift to multiply or divide by 2, e.g. 5 >> 1 = 2 (floor division), 5 << 1 = 10

```python
AND = n & 1
OR = n | 1
XOR = n ^ 1
NOT = ~n
Left Shift = n << 1
Right Shift = n >> 1
```
