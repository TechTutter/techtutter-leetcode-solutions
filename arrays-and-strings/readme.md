# Arrays And Strings

### Easily Truncate an Array in O(1)

To remove elements from the array, you can set the length to a lower value and truncate it to dereference them.

```typescript
const x = [0, 1, 2, 3, 4, 5];
x.length = 2;
console.log(x); // -> [0,1]
```
